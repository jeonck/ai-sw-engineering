м •лӢөмқ„ м–»мңјл Өл©ҙ мҳӨлӢөмқ„ кІҢмӢңн•ҳлқј, Cunninghamмқҳ лІ•м№ҷ

I. м •ліҙ мқём¶ңмқҳ м—ӯм„Өм Ғ кІҪлЎң, Cunninghamмқҳ лІ•м№ҷ к°ңмҡ”вҖӢ

м •мқҳ: "мқён„°л„·м—җм„ң мҳ¬л°”лҘё лӢөмқ„ м–»лҠ” к°ҖмһҘ мўӢмқҖ л°©лІ•мқҖ м§Ҳл¬ёмқ„ н•ҳлҠ” кІғмқҙ м•„лӢҲлқј, нӢҖлҰ° лӢөмқ„ мҳ¬лҰ¬лҠ” кІғмқҙлӢӨ"лқјлҠ” мқён„°л„· л¬ёнҷ”мқҳ н–үлҸҷ м§Җм№Ёмқҙмһҗ мӢ¬лҰ¬м Ғ лІ•м№ҷ

нҠ№м§•:
( көҗм • ліёлҠҘ нҷңмҡ© ) мӮ¬лһҢл“ӨмқҖ м§Ҳл¬ём—җ лӢөн•ҳлҠ” кІғліҙлӢӨ нғҖмқёмқҳ мҳӨлҘҳлҘј л°”лЎңмһЎлҠ” лҚ° нӣЁм”¬ лҚ” к°•л Ҙн•ң м§Җм Ғ лҸҷкё°лҘј лҠҗлӮҢ
( м •ліҙмқҳ м •көҗнҷ” ) мҳӨлӢөмқ„ л№„нҢҗн•ҳлҠ” кіјм •м—җм„ң м „л¬ёк°Җл“ӨмқҖ лӢЁмҲң лӢөліҖліҙлӢӨ нӣЁм”¬ мғҒм„ён•ҳкі  к№Ҡмқҙ мһҲлҠ” м§ҖмӢқмқ„ мҸҹм•„лӮҙлҠ” кІҪн–Ҙмқҙ мһҲмқҢ
( м§‘лӢЁ м§Җм„ұ к°ҖмҶҚ ) м»Өл®ӨлӢҲнӢ°мқҳ мҶҢк·№м Ғмқё нғңлҸ„лҘј м Ғк·№м Ғмқё к°ңмһ…мңјлЎң м „нҷҳмӢңмјң л¬ём ң н•ҙкІ°мқҳ мҶҚлҸ„лҘј нҡҚкё°м ҒмңјлЎң лҶ’мһ„

II. Cunninghamмқҳ лІ•м№ҷмқҳ мһ‘лҸҷ л©”м»ӨлӢҲмҰҳкіј нҳ•мғҒнҷ”вҖӢ

к°Җ. нғҖмқёмқҳ мқём§Җм Ғ к°ңмһ… мң лҸ„ л°Ҹ м •ліҙ нҡҚл“қ лӘЁлҚёвҖӢ

лӮҳ. м§Ҳл¬ён•ҳкё° vs мҳӨлӢө кІҢмӢңн•ҳкё° л№„көҗвҖӢ

л№„көҗ н•ӯлӘ©м „нҶөм Ғ м§Ҳл¬ё (Asking)Cunninghamмқҳ лІ•м№ҷ (Wrong Answer)
м°ём—¬ лҸҷкё°мҲңмҲҳн•ң лҸ„мӣҖ/мқҙнғҖмӢ¬ мқҳмЎҙкөҗм • ліёлҠҘ л°Ҹ м§Җм Ғ кіјмӢңмҡ• мһҗк·№
м •ліҙмқҳ к№Ҡмқҙм§Ҳл¬ёмқҳ мҲҳмӨҖм—җ л§һм¶ҳ лӢЁнҺём Ғ м •ліҙмҳӨлӢөмқ„ л°ҳл°•н•ҳкё° мң„н•ң мӢ¬мёөм Ғ л…јкұ° нҸ¬н•Ё
л°ҳмқ‘ мҶҚлҸ„мғҒлҢҖм ҒмңјлЎң лҠҗлҰј (кҙҖмӢ¬ л¶ҖмЎұ)л§Өмҡ° л№ лҰ„ (мҰүк°Ғм Ғмқё л°ҳл°ңмӢ¬ мң л°ң)
лҰ¬мҠӨнҒ¬л¬ҙмқ‘лӢө лҰ¬мҠӨнҒ¬л№„лӮң л°Ҹ нҸүнҢҗ м Җн•ҳ лҰ¬мҠӨнҒ¬ (к°ҖлӘ…м„ұ н•„мҡ”)

III. мҶҢн”„нҠёмӣЁм–ҙ мғқнғңкі„м—җм„ңмқҳ Cunninghamмқҳ лІ•м№ҷ м Ғмҡ© м „лһөвҖӢ

к°Җ. мҳӨн”ҲмҶҢмҠӨ л°Ҹ кё°мҲ  м»Өл®ӨлӢҲнӢ° нҷңмҡ© л°©м•ҲвҖӢ

м Ғмҡ© мҳҒм—ӯмғҒм„ё лӮҙмҡ©кё°лҢҖ нҡЁкіј
Stack Overflow"мқҙ л°©лІ•мқҙ мөңм„ мһ…лӢҲлӢӨ"лқјкі  нҡЁмңЁ лӮ®мқҖ мҪ”л“ң кІҢмӢңкі мҲҳл“Өмқҳ лҰ¬нҢ©нҶ л§Ғ л°Ҹ м„ұлҠҘ мөңм Ғнҷ” мҪ”л“ң мң лҸ„
Open Sourceл¶Ҳмҷ„м „н•ҳм§Җл§Ң мһ‘лҸҷн•ҳлҠ” Draft PR м ңм¶ңл©”мқён…Ңмқҙл„Ҳл“Өмқҳ кө¬мІҙм Ғмқё м„Өкі„ к°Җмқҙл“ңлқјмқё нҷ•ліҙ
Tech DesignнҡҢмқҳ мӢң "мқҙл ҮкІҢ м„Өкі„н•ҳл©ҙ мҷ„лІҪн•ҳмЈ ?"лқјкі  лҸ„л°ңнҢҖмӣҗл“Өмқҳ мҲЁкІЁм§„ м—Јм§Җ мјҖмқҙмҠӨ л°Ҹ л°ҳлҢҖ л…јкұ° лҸ„м¶ң

лӮҳ. мӢӨл¬ҙм Ғ м Ғмҡ© мӢңмӮ¬м җвҖӢ

  • Ethical Boundary: мқҳлҸ„м Ғмқё кё°л§ҢліҙлӢӨлҠ” "лӮҙк°Җ м•Ңкё°лЎңлҠ” мқҙл ҮлӢӨ"лҠ” мӢқмқҳ мһҗмӢ к°җ мһҲлҠ” к°Җм„Ө м ңмӢң нҳ•мӢқмқ„ м·Ён•ҳм—¬ м»Өл®ӨлӢҲнӢ°мқҳ н”јлЎңлҸ„лҘј лӮ®м¶°м•ј н•Ё
  • Rubber Ducking Extension: нҳјмһҗм„ң кі лҜјн•  л•ҢліҙлӢӨ нғҖмқём—җкІҢ 'м„ӨлӘ…'н•ҳкұ°лӮҳ 'мЈјмһҘ'н•  л•Ң мһҗмӢ мқҳ л…јлҰ¬м Ғ н—Ҳм җмқҙ лҚ” лӘ…нҷ•нһҲ л“ңлҹ¬лӮЁ
  • Psychological Safety: нҢҖ лӮҙм—җм„ң м»ӨлӢқн–„мқҳ лІ•м№ҷмқҙ мһ‘лҸҷн•ҳл Өл©ҙ, нӢҖлҰ° мқҳкІ¬мқ„ лӮҙлҚ”лқјлҸ„ л№„лӮңл°ӣм§Җ м•ҠлҠ” мӢ¬лҰ¬м Ғ м•Ҳм „к°җмқҙ м„ н–үлҗҳм–ҙм•ј н•Ё (Hanlonмқҳ л©ҙлҸ„лӮ  м—°кі„)
