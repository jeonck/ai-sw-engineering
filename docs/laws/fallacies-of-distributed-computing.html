л¶„мӮ° мӢңмҠӨн…ң м„Өкі„мқҳ 8к°Җм§Җ н•Ём •, л¶„мӮ° м»ҙн“ЁнҢ…мқҳ мҳӨлҘҳ (Fallacies of Distributed Computing)

I. мқҙмғҒм Ғ л„ӨнҠёмӣҢнҒ¬м—җ лҢҖн•ң нҷҳмғҒкіј мӢӨм ң, л¶„мӮ° м»ҙн“ЁнҢ… мҳӨлҘҳмқҳ к°ңмҡ”вҖӢ

м •мқҳ : л¶„мӮ° мӢңмҠӨн…ңмқ„ мІҳмқҢ м„Өкі„н•ҳлҠ” к°ңл°ңмһҗл“Өмқҙ л„ӨнҠёмӣҢнҒ¬мқҳ л¬јлҰ¬м Ғ нҠ№м„ұмқ„ к°„кіјн•ҳкі  лІ”н•ҳкё° мү¬мҡҙ 8к°Җм§Җ мһҳлӘ»лҗң к°Җм •( Fallacies )

н•өмӢ¬ нҠ№м§• л°Ҹ мӢңмӮ¬м җ :
( л¬јлҰ¬м Ғ н•ңкі„ мқём • ) л„ӨнҠёмӣҢнҒ¬лҠ” кІ°мҪ” мҷ„лІҪн•ҳм§Җ м•Ҡмңјл©°, м§Җм—°, лӢЁм Ҳ, лҢҖм—ӯнҸӯ м ңн•ң л“ұ нҳ„мӢӨм Ғмқё м ңм•Ҫ мЎ°кұҙмқ„ м„Өкі„мқҳ мғҒмҲҳлЎң м·Ёкёүн•ҙм•ј н•Ё
( мһҘм•  лӮҙм„ұ м„Өкі„ ) "л„ӨнҠёмӣҢнҒ¬лҠ” н•ӯмғҒ лҜҝмқ„ мҲҳ мһҲлӢӨ"лҠ” к°Җм •мқ„ лІ„лҰ¬кі , мһҘм•  л°ңмғқмқ„ м „м ңлЎң н•ң м„ңнӮ· лёҢл Ҳмқҙм»Ө( Circuit Breaker ) л“ұ л°©м–ҙм Ғ м„Өкі„ м Ғмҡ©
( L. Peter Deutschмқҳ нҶөм°° ) мҚ¬ л§ҲмқҙнҒ¬лЎңмӢңмҠӨн…ңмҰҲмқҳ м—”м§ҖлӢҲм–ҙл“Өмқҙ м ңм•Ҳн•ң мқҙ мӣҗм№ҷмқҖ нҳ„лҢҖ нҒҙлқјмҡ°л“ң л„ӨмқҙнӢ°лёҢ м•„нӮӨн…ҚмІҳмҷҖ MSA м„Өкі„мқҳ н•„мҲҳ м§Җм№Ёмқҙ лҗЁ
( ліҙм•Ҳ л°Ҹ мӢ лў°м„ұ м§ҒкІ° ) л„ӨнҠёмӣҢнҒ¬мқҳ л¶ҲнҲ¬лӘ…м„ұкіј мң„нҳ‘мқ„ мқём •н•ҳлҠ” кІғм—җм„ңл¶Җн„° м ңлЎң нҠёлҹ¬мҠӨнҠё( Zero Trust ) ліҙм•Ҳ лӘЁлҚёмқҙ мӢңмһ‘лҗЁ

II. л¶„мӮ° м»ҙн“ЁнҢ…мқҳ 8к°Җм§Җ мҳӨлҘҳ мғҒм„ё л¶„м„қвҖӢ

к°Җ. 8к°Җм§Җ мҳӨлҘҳ лҰ¬мҠӨнҠё л°Ҹ нҳ„мӢӨм Ғ л°ҳлЎ вҖӢ

мҲңлІҲк°ңл°ңмһҗмқҳ мһҳлӘ»лҗң к°Җм • (The Fallacy)нҳ„мӢӨм—җм„ңмқҳ мӢӨмІҙ (The Reality)
1л„ӨнҠёмӣҢнҒ¬лҠ” мӢ лў°н•  мҲҳ мһҲлӢӨ.нҢЁнӮ·мқҖ мң мӢӨлҗҳл©°, м—°кІ°мқҖ мҳҲкі  м—Ҷмқҙ лҒҠкёҙлӢӨ.
2м§Җм—° мӢңк°„(Latency)мқҖ 0мқҙлӢӨ.лҚ°мқҙн„° м „мҶЎм—җлҠ” л°ҳл“ңмӢң л¬јлҰ¬м Ғ мӢңк°„мқҙ мҶҢмҡ”лҗңлӢӨ.
3лҢҖм—ӯнҸӯ(Bandwidth)мқҖ л¬ҙн•ңн•ҳлӢӨ.л„ӨнҠёмӣҢнҒ¬ мһҗмӣҗмқҖ н•ңм •лҗҳм–ҙ мһҲмңјл©° лі‘лӘ©мқҙ л°ңмғқн•ңлӢӨ.
4л„ӨнҠёмӣҢнҒ¬лҠ” м•Ҳм „н•ҳлӢӨ.лҚ°мқҙн„°лҠ” лҸ„мІӯ, ліҖмЎ°лҗ  мҲҳ мһҲмңјл©° мң„нҳ‘м—җ л…ём¶ңлҗҳм–ҙ мһҲлӢӨ.
5нҶ нҸҙлЎңм§Җ(Topology)лҠ” ліҖн•ҳм§Җ м•ҠлҠ”лӢӨ.л…ёл“ңлҠ” м¶”к°Җ/мӮӯм ңлҗҳл©° л„ӨнҠёмӣҢнҒ¬ кІҪлЎңлҠ” лҸҷм ҒмңјлЎң ліҖн•ңлӢӨ.
6кҙҖлҰ¬мһҗлҠ” н•ң лӘ…мқҙлӢӨ.лӢӨмҲҳмқҳ кҙҖлҰ¬мһҗмҷҖ м •мұ…мқҙ м¶©лҸҢн•ҳл©° нҶөм ңк°Җ м–ҙл өлӢӨ.
7мҡҙмҶЎ л№„мҡ©(Transport Cost)мқҖ 0мқҙлӢӨ.лҚ°мқҙн„° м§Ғл ¬нҷ”, м—ӯм§Ғл ¬нҷ” л°Ҹ м „мҶЎм—җ мһҗмӣҗмқҙ мҶҢлӘЁлҗңлӢӨ.
8л„ӨнҠёмӣҢнҒ¬лҠ” к· мқјн•ҳлӢӨ(Homogeneous).лӢӨм–‘н•ң н”„лЎңнҶ мҪң, мһҘл№„, м„ұлҠҘмқҳ л…ёл“ңк°Җ нҳјмһ¬н•ңлӢӨ.

лӮҳ. мҳӨлҘҳ к°„мқҳ мғҒнҳё мһ‘мҡ© л°Ҹ мҳҒн–ҘлҸ„вҖӢ

III. л¶„мӮ° м»ҙн“ЁнҢ… мҳӨлҘҳ к·№ліөмқ„ мң„н•ң м—”м§ҖлӢҲм–ҙл§Ғ м „лһөвҖӢ

к°Җ. мҳӨлҘҳ лҢҖмқ‘мқ„ мң„н•ң кё°мҲ м Ғ н•ҙкІ°мұ… л№„көҗвҖӢ

к·№ліө м „лһөкҙҖл Ё мҳӨлҘҳ н•ӯлӘ©кө¬мІҙм Ғ кө¬нҳ„ кё°мҲ 
мһ¬мӢңлҸ„ л°Ҹ нғҖмһ„м•„мӣғ1, 2Exponential Backoff, Retry Policy
л№„лҸҷкё° нҶөмӢ 2, 3Message Queue (Kafka, RabbitMQ), Pub/Sub
мғҒнҳё мқёмҰқ л°Ҹ м•”нҳёнҷ”4mTLS, TLS 1.3, IPsec
м„ңл№„мҠӨ л””мҠӨм»ӨлІ„лҰ¬5, 8Kubernetes Service, Consul, Istio
лҚ°мқҙн„° м••м¶•/мөңм Ғнҷ”3, 7gRPC (Protocol Buffers), Avro, MessagePack

лӮҳ. мӢӨл¬ҙм Ғ м„Өкі„ к°Җмқҙл“ңлқјмқёвҖӢ

  • кҙҖмёЎ к°ҖлҠҘм„ұ(Observability) нҷ•ліҙ : л„ӨнҠёмӣҢнҒ¬ м§Җм—°кіј мҳӨлҘҳлҘј мӢӨмӢңк°„мңјлЎң лӘЁлӢҲн„°л§Ғн•ҳм—¬ к°Җм •мқҙ нӢҖл ёмқҢмқ„ лҚ°мқҙн„°лЎң нҷ•мқё( Prometheus, Jaeger )
  • л„ӨнҠёмӣҢнҒ¬ л¶ҲмӢ (Trust No One) : л„ӨнҠёмӣҢнҒ¬ лӮҙл¶ҖлқјлҸ„ кІ°мҪ” м•Ҳм „н•ҳм§Җ м•ҠлӢӨлҠ” м „м ңн•ҳм—җ лӘЁл“  нҶөмӢ м—җ лҢҖн•ҙ мқёмҰқкіј мқёк°Җ мҲҳн–ү( Zero Trust )
  • мӢңмҠӨн…ң лҸ…лҰҪм„ұ мң м§Җ : нҠ№м • кҙҖлҰ¬мһҗлӮҳ нҠ№м • мһҘл№„мқҳ мЎҙмһ¬м—җ мқҳмЎҙн•ҳм§Җ м•ҠлҸ„лЎқ лҠҗмҠЁн•ң кІ°н•©( Decoupling ) л°Ҹ н‘ңмӨҖ н”„лЎңнҶ мҪң м§Җн–Ҙ

н•өмӢ¬ : л¶„мӮ° м»ҙн“ЁнҢ…мқҳ мҳӨлҘҳлҘј мқҙн•ҙн•ҳлҠ” кІғмқҖ л¶„мӮ° мӢңмҠӨн…ң м„Өкі„мқҳ мІ«кұёмқҢмқҙл©°, л„ӨнҠёмӣҢнҒ¬мқҳ л¶Ҳнҷ•мӢӨм„ұмқ„ мҶҢн”„нҠёмӣЁм–ҙ лЎңм§ҒмңјлЎң к·№ліөн•  л•Ң л№„лЎңмҶҢ м§„м •н•ң нҒҙлқјмҡ°л“ң л„ӨмқҙнӢ°лёҢ мӢңмҠӨн…ңмқҙ мҷ„м„ұлҗЁ
